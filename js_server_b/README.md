# The initital setup
```bash
 $ cd <project_directory>
 $ cd js_server_b
 $ npm init -y
 $ npm install fs yaml fast-xml-parser csv-parser
 $ npm i express
```
Add key-value pair "type": "module" in 'package.json'.
Created index.js and all other needed files

# Install
```bash
 $ cd <project_directory>
 $ cd js_server_b
 $ npm install
```

# Usage
```bash
 $ cd <project_directory>
 $ cd js_server_b
 $ node index.js
or
 $ nodemon index.js
```
